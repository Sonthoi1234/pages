Update logs

This page contains update logs for our Tower Defense Game "extension", whose main purposes include documenting bugfixes, new features/changes and more. Contains mostly major updates, but can also have beta versions packed in.
Bugs shown in some update logs will be fixed as soon as possible. If we don't fix it, just know that we might have included it as a intended feature, and if we did then don't request fixing it.
Some updates might not be here so if you want us to write it here, give us some information about what that update added (cause we don't have big enough brains to remember it all), and we'll try our best!
We do not receive requests for adding update logs for versions older than 1.6.0.

Version 1.9.0 (november 25th beta) Beta release

One of the load of beta releases for the new major update of this game. We've been waiting for this in a long time.

New features, changes to them and bug fixes:

  • New enemy types: Hypersonic and Balloon.
  • New turrets: Knife (previously called Slash) and Plasma (upcoming).
  • Changes to the developer console (requires special code to unlock).
  • Massive overhaul to how explosions work and how enemies receive effect damage.
  • Special upgrades for Electric turret.
  • A more updated How To Play menu.
  • Some random bugs were fixed.
Balance changes were made, such as:
  • Redesigned textures for the map selection menu background and lives counter.
  • Regen boss ability buffed.
There were also some notable bugfixes, some being:
  • Snowstorm effect stacking due to clones.
  • Enemies dying immediately after spawn, making you lose lives (can still happen with huge lag).

Version 1.8.0 (april 15th beta) Major release

This is the official release for TDG 1.8.0.
This version adds a lot of new stuffs, such as:

  • New TNT & Fungi turret (see here for information).
  • Regen boss added (see here for information).
  • Special upgrades for Fire, Snow, and TNT turret (see here for information).
  • An in-game How To Play menu.
Changes were made to the game, such as:
  • Redesigned textures for the map selection menu background and lives counter.
  • Regen boss ability buffed.
There were also some notable bugfixes, some being:
  • Snowstorm effect stacking due to clones.
  • Enemies dying immediately after spawn, making you lose lives (can still happen with huge lag).
