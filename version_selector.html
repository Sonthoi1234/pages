Beta 1.8.1 (july 7th beta) Beta release

The last beta for version 1.8.1, and can also be considered the official release of it.
A new map category was added, named Snow (there are only 2 for now). All turrets shoot 20% slower, but snowstorms from Snow turret last twice as long.

Version 1.8.0 (official) Major release

The official release for TDG 1.8.0!
I'm too lazy to go through all the details, so you'll have to play the game yourself and see all of them in the "Changelogs" menu (title screen)
Notable changes include:

  • A proper How To Play menu.
  • Fungi turret added.
  • More special upgardes & abilities for Snow & TNT turrets.

Version 1.6.2 Small patch

An older version of the game, mainly an update for TDG 1.6.0.
This version does not support turret selling and didn't have Forest maps, so it was kinda empty.

Version 1.6.2 (fps option test) Beta release

The second beta for TDG 1.6.0 (the last being 1.6.3, but we won't talk about that now)
This version added options for switching between 30fps, 60fps and 120fps. Note that high framerates are unstable, so be aware!
Upgrades don't work for some reason, but you should be able to play it still.

Version 1.3.0 Major release

This update is pretty old, so don't expect this to have many features yet.
The update added Gear and Electric turrets, which are still in the game to this day.
Gear turret was broken after 1.6.0 (where upgrades were added), but the bug was fixed after the latest 1.8.0 beta came out

Version 1.0.0 Major release

This is the oldest and also the first version of the game.
Not much to say here, since it doesn't have a lot of features.
If you know that Wind turret is a meta right now then after playing this version, you might want to change your mind and stop becoming a noob.
